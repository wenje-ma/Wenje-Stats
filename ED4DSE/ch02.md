## 2.2 多项式插值

### 2.2.1 等间距节点的 Runge 现象

给定 $n$ 个等间距点 $D_i = \frac{i-1}{n-1}, i=1,\cdots,n$，构造 $n-1$ 次插值多项式。

**Chebyshev 多项式基底**（将 $x \in [0,1]$ 映射到 $d \in [-1,1]$）：

令 $d_i = 2D_i - 1$，设计矩阵为：

$$T_{ij} = \cos[(j-1)\arccos(d_i)], \quad i,j = 1,\cdots,n$$

**问题**：等间距点在高次多项式插值时会产生剧烈的边界振荡（Runge 现象）。

### 2.2.2 Chebyshev 节点

Chebyshev 节点在 $[-1,1]$ 上的定义为：

$$d_i = \cos\left(\frac{2i-1}{2n}\pi\right), \quad i = 1,\cdots,n$$

映射到 $[0,1]$：$D_i = (d_i + 1)/2$

**优点**：节点在两端更密集，有效抑制 Runge 现象。

### 代码实现

```python
def chebyshev_interp_coeffs(D, y):
    """计算 Chebyshev 多项式插值系数"""
    d = 2 * D - 1  # 映射到 [-1, 1]
    n = len(D)
    X = np.column_stack([np.cos(j * np.arccos(d)) for j in range(n)])
    return np.linalg.solve(X, y)

def chebyshev_eval(a, u):
    """用 Chebyshev 系数计算预测值"""
    us = 2 * u - 1
    n = len(a)
    T = np.column_stack([np.cos(j * np.arccos(us)) for j in range(n)])
    return T @ a
```

## 2.3 RBF 插值 (Radial Basis Function)

RBF 插值使用径向基函数作为核：

$$R_{ij} = \exp\left[-\left(\frac{|D_i - D_j|}{\theta}\right)^2\right]$$

系数通过求解线性系统得到：$R\alpha = y$

对新点 $x^*$ 的预测：$\hat{y}(x^*) = \sum_{i=1}^n \alpha_i \exp\left[-\left(\frac{|x^* - D_i|}{\theta}\right)^2\right]$

**$\theta$ 的影响**：
- $\theta = 0.01$：核非常窄，每个点只影响很小的邻域 → 过拟合，振荡严重
- $\theta = 0.1$：适中的核宽度，平衡拟合与平滑
- $\theta = 1.0$：核很宽，所有点相互影响 → 过于平滑，欠拟合

```python
def rbf_interp(D, y, theta, test):
    """RBF 插值"""
    E = np.abs(D[:, None] - D[None, :])
    R = np.exp(-((E / theta) ** 2))
    coef = np.linalg.solve(R, y)
    M = np.exp(-(((test[:, None] - D[None, :]) / theta) ** 2))
    return M @ coef
```

## 2.4 普通克里金 (Ordinary Kriging)

### 模型

假设 $y(x) = \mu + z(x)$，其中：
- $\mu$ 是未知常数均值
- $z(x)$ 是零均值高斯过程，协方差为 $Cov(z(x_i), z(x_j)) = \tau^2 R(x_i, x_j)$

### 公式

相关矩阵：$R_{ij} = \exp\left[-\left(\frac{|D_i-D_j|}{\theta}\right)^2\right]$

均值估计：$\hat{\mu} = \frac{\mathbf{1}^T R^{-1} y}{\mathbf{1}^T R^{-1} \mathbf{1}}$

过程方差：$\hat{\tau}^2 = \frac{(y - \hat{\mu}\mathbf{1})^T R^{-1}(y - \hat{\mu}\mathbf{1})}{n}$

预测均值：$\hat{y}(x^*) = \hat{\mu} + r(x^*)^T R^{-1}(y - \hat{\mu}\mathbf{1})$

预测方差：

$$s^2(x^*) = \tau^2\left[1 - r^T R^{-1} r + \frac{(1 - \mathbf{1}^T R^{-1} r)^2}{\mathbf{1}^T R^{-1} \mathbf{1}}\right]$$

其中 $r_i(x^*) = \exp\left[-\left(\frac{|x^*-D_i|}{\theta}\right)^2\right]$

### 代码实现

```python
def ordinary_kriging(D, y, theta, test):
    """普通克里金插值"""
    n = len(D)
    E = np.abs(D[:, None] - D[None, :])
    R = np.exp(-((E / theta) ** 2))
    one = np.ones(n)
    Rinv = np.linalg.inv(R + 1e-10 * np.eye(n))
    
    # 估计均值
    mu = float((one @ Rinv @ y) / (one @ Rinv @ one))
    coef = Rinv @ (y - mu)
    
    # 估计过程方差
    tau2 = float((y - mu).T @ Rinv @ (y - mu) / n)
    
    # 预测
    M = np.exp(-(((test[:, None] - D[None, :]) / theta) ** 2))
    mean = mu + M @ coef
    
    # 预测方差
    fac = 1.0 / float(one @ Rinv @ one)
    v = []
    for i in range(len(test)):
        r = M[i, :]
        term = 1.0 - (r @ Rinv @ r) + fac * (1.0 - (r @ Rinv @ one)) ** 2
        v.append(tau2 * term)
    sd = np.sqrt(np.maximum(v, 0.0))
    return mean, sd
```

## 2.5 高斯过程 (Gaussian Process) 可视化

### 先验分布

在没有任何数据的情况下，GP 先验认为函数值服从：

$$f(x) \sim \mathcal{GP}(0, k(x, x'))$$

从先验中采样可以得到随机的函数曲线，反映了我们对函数的先验认知。

### 后验分布

加入观测数据后，条件后验为：

$$f(x^*) | D, y \sim N(\hat{f}(x^*), s^2(x^*))$$

后验样本在数据点处被"钉住"（通过数据约束），远离数据点处不确定性大。

### 三面板图

1. **先验分布**：多条随机函数曲线，在 $x=0.5$ 附近散开
2. **数据**：在先验上叠加观测数据点
3. **后验分布**：经过数据更新后，函数曲线在数据点处收敛

## 2.6 多项式回归

### 3 次多项式回归

模型：$y = \beta_0 + \beta_1 x + \beta_2 x^2 + \beta_3 x^3 + \varepsilon$

使用 Chebyshev 多项式基（$d = 2x - 1$）：

$$y = \beta_0 + \beta_1 d + \beta_2 d^2 + \beta_3 d^3 + \varepsilon$$

### 8 次多项式回归

模型：$y = \sum_{k=0}^8 \beta_k d^k + \varepsilon$

### D-最优设计 vs 等间距设计

**D-最优设计**：最大化信息矩阵的行列式 $\det(X^T X)$，从而最小化参数估计的广义方差。

等间距设计在高次多项式回归中效果差，因为边界点不够密集。

## 2.7 核岭回归 (Kernel Ridge Regression)

### 模型

$$\hat{y}(x^*) = k(x^*)^T (K + \lambda I)^{-1} y$$

其中 $K_{ij} = \exp\left[-\left(\frac{|D_i-D_j|}{\theta}\right)^2\right]$ 是核矩阵，$\lambda$ 是正则化参数。

### 超参数选择：MSCV

使用修正的留一交叉验证 (Modified Score Cross-Validation)：

$$\text{MSCV}(\theta, \lambda) = \log\left(\frac{1}{n}\sum_{i=1}^n \left(\frac{e_i}{a_{ii}}\right)^2\right)$$

其中 $e = (K + \lambda I)^{-1} y$，$a_{ii}$ 是 $(K + \lambda I)^{-1}$ 的对角线元素。

```python
def kernel_ridge_cv(D, y, test):
    """核岭回归带 MSCV 超参数调优"""
    E = np.abs(D[:, None] - D[None, :])
    n_obs = len(D)
    
    def mscv(para):
        theta, lam = para[0], para[1]
        R = np.exp(-((E / theta) ** 2))
        Rinv = np.linalg.solve(R + lam * np.eye(n_obs), np.eye(n_obs))
        cv = (Rinv @ y) / np.diag(Rinv)
        return np.log(np.mean(cv ** 2))
    
    result = minimize(mscv, x0=[0.1, 0.1], 
                      bounds=[(0.05, 1.0), (0.001, 1.0)], 
                      method="L-BFGS-B")
    theta, lam = result.x
    
    R = np.exp(-((E / theta) ** 2))
    coef = np.linalg.solve(R + lam * np.eye(n_obs), y)
    M = np.exp(-(((test[:, None] - D[None, :]) / theta) ** 2))
    return M @ coef
```

## 📝 例题详解

### 例题 2.1：Chebyshev 节点的推导

**问题**：证明 Chebyshev 节点 $d_i = \cos\left(\frac{2i-1}{2n}\pi\right)$ 是 $[-1,1]$ 上使 $n$ 次多项式插值误差最小化的节点。

**解**：
对于 $n-1$ 次多项式插值，误差公式为：

$$|f(x) - P_{n-1}(x)| \leq \frac{|f^{(n)}(\xi)|}{n!} \prod_{i=1}^n |x - x_i|$$

Chebyshev 节点最小化 $\max_{x\in[-1,1]} \prod_{i=1}^n |x - x_i|$，从而使误差上界最小化。

### 例题 2.2：RBF 参数选择

**问题**：解释为什么 RBF 插值的 $\theta$ 参数不能太小也不能太大。

**解**：
- 当 $\theta \to 0$ 时，$R_{ij} \to 0$（$i\neq j$），$R_{ii}=1$，系数 $\alpha_i = y_i$，预测在数据点之间迅速衰减到0 → 过拟合
- 当 $\theta \to \infty$ 时，$R_{ij} \to 1$，矩阵近乎奇异 → 预测过于平滑
- 最优 $\theta$ 通常在数据点间距的量级附近

### 例题 2.3：多项式回归的 D-最优设计

**问题**：对 8 次多项式回归，为什么 D-最优设计与等间距设计不同？D-最优设计有什么特点？

**解**：
- 等间距设计在边界处只有少量点，对高次项估计不充分
- D-最优设计会更多地在边界处布点（与 Chebyshev 节点类似）
- 这是因为高次多项式在边界处的基函数变化更快，需要更密集的点才能准确估计系数

## 💡 核心要点

1. **Chebyshev 节点** 有效解决了多项式插值的 Runge 现象
2. **RBF 插值** 的关键是选择合适的 $\theta$（核宽度）
3. **普通克里金** 在预测的同时提供不确定性估计
4. **高斯过程** 是一种灵活的非参数贝叶斯方法
5. **D-最优设计** 比等间距设计更适合高次多项式回归
6. **核岭回归** 结合了核方法和正则化，具有很好的泛化性能
