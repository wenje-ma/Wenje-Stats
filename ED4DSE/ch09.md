## 9.1 模型校准概述

**问题**：计算机模型 $y = \eta(x, \theta)$ 依赖于未知参数 $\theta$，如何设计实验来估计 $\theta$？

**两阶段**：
1. 实验设计：选择实验输入点 $x_1, \cdots, x_n$
2. 参数估计：用实验数据估计 $\theta$

---

## 9.2 SO2 非线性回归模型

### 模型

$$h(u, \eta) = y_0 + \left(\frac{\beta_0 + \beta_1 x_1}{\beta_0 + \beta_1 x_1 + x_2} - y_0\right) \left[1 - \exp\left(-\frac{(\beta_0 + \beta_1 x_1 + x_2) \cdot dt}{m}\right)\right]$$

其中 $u = [x_1, x_2, dt, y_0]$ 是输入，$\eta = [\beta_0, \beta_1, m]$ 是待校准参数。

```python
def h_so2(u, eta):
    """SO2 模型函数"""
    x1, x2, dt, y0 = u
    beta0, beta1, m = eta
    rate = beta0 + beta1 * x1 + x2
    frac = (beta0 + beta1 * x1) / rate
    val = y0 + (frac - y0) * (1 - np.exp(-rate * dt / m))
    return val
```

### 参数估计 (最小二乘)

$$\hat{\eta} = \arg\min_\eta \sum_{i=1}^n (y_i - h(u_i, \eta))^2$$

```python
def sse(eta):
    yhat = np.array([h_so2(X[i], eta) for i in range(len(X))])
    return np.sum((y - yhat) ** 2)

result = minimize(sse, x0=[0, 1, 1000], method="Nelder-Mead")
eta_hat = result.x
```

### Fisher 信息矩阵 (FIM)

$$F(\eta) = S(\eta)^T S(\eta)$$

其中 $S$ 是灵敏度矩阵，$S_{ij} = \frac{\partial h(u_i, \eta)}{\partial \eta_j}$。

```python
def gradient_so2(x, eta):
    """SO2 模型对参数的梯度"""
    x1, x2, dt, y0 = x
    beta0, beta1, m = eta
    rate = beta0 + beta1 * x1 + x2
    frac = (beta0 + beta1 * x1) / rate
    term1 = frac - y0
    term2 = np.exp(-rate * dt / m)
    
    v1 = term1 * term2 * dt / m + x2 / rate**2 * (1 - term2)
    v2 = term1 * term2 * x1 * dt / m + x1 * x2 / rate**2 * (1 - term2)
    v3 = -term1 * term2 * rate * dt / m**2
    
    return np.array([v1, v2, v3])
```

## 9.3 最优设计理论

### D-最优设计

选择设计 $D$ 最大化 $\det(F(D, \eta))$，即最小化参数估计的广义方差。

**局部 D-最优**：在参数 $\eta_0$ 的某个猜测值处优化。

**贝叶斯 D-最优**：在参数的先验分布 $p(\eta)$ 上平均：

$$D_B = \arg\max_D \int \log\det(F(D, \eta)) p(\eta) d\eta$$

### ICAOD 算法

ICAOD (Iterative Coordinate Ascent Optimal Design) 通过迭代更新设计点和权重：

```python
# 通过 R 的 ICAOD 包计算
```

- 贝叶斯 D-最优：设计点更分散，权重更均匀（对参数不确定性更稳健）

## 9.5 稳健设计 (Robust Design)

### 局部 vs 稳健设计

当参数 $\theta$ 不确定时（只知道先验分布），局部 D-最优设计可能过于依赖参数猜测。

**稳健设计方法**：
1. 从参数的后验/先验分布中采样多个 $\theta$ 值
2. 对每个 $\theta$ 计算最优设计
3. 用支持点选择一组对所有 $\theta$ 值都有效的设计

```python
# 候选设计：大量 LHD 点
# 对每个候选点计算其在所有参数样本下的性能
# 选择平均性能最好的 n 个点
D_robust = out["robust_design.csv"].values
```

### 增广设计 (Augmentation)

已有 $n_0$ 个初始实验点，需要用 $n_{\text{new}}$ 个新点增广。

**MaxPro 增广**：在已有设计的基础上，用 MaxPro 准则选择新点。

## 📝 例题详解

### 例题 9.1：FIM 与参数估计精度

**问题**：为什么 FIM 的行列式越大，参数估计越精确？

**解**：
对于非线性最小二乘，参数估计的协方差矩阵近似为：

$$\text{Cov}(\hat{\eta}) \approx \sigma^2 [F(\eta)]^{-1} = \sigma^2 [S^T S]^{-1}$$

因此 $\det(\text{Cov}(\hat{\eta})) \propto 1/\det(F)$。最大化 $\det(F)$ 等价于最小化估计参数的广义方差（置信椭球体积）。

### 例题 9.2：局部 vs 贝叶斯 D-最优

**问题**：局部 D-最优设计依赖参数猜测值 $\eta_0$，如果 $\eta_0$ 不准确会怎样？

**解**：
- 如果猜测准确，局部 D-最优设计在 $\eta_0$ 处效率最高
- 如果猜测不准确，局部设计可能非常低效
- 贝叶斯 D-最优通过对先验分布积分，对参数不确定性更稳健
- 实际中，先用一个便宜的实验获得 $\eta$ 的合理估计，再设计精确实验

### 例题 9.3：功能输出实验设计

**问题**：在功能输出实验中，应该在哪些时间点测量？

**解**：
功能输出 Kriging 选择的时间点通常：
1. 在函数变化快的区域（导数大）密集
2. 在不确定性大的区域密集
3. 对应于模型参数的灵敏度高的位置

## 💡 核心要点

1. **模型校准**的核心是设计实验以精确估计模型参数
2. **FIM** 连接了设计矩阵和参数估计精度
3. **局部 D-最优**依赖参数猜测，**贝叶斯 D-最优**更稳健
4. **ICAOD** 是一种有效的数值最优设计算法
5. **功能输出 Kriging** 处理输出为函数的情况
6. **稳健设计**通过在参数先验上平均来抵御不确定性
7. **增广设计**在已有设计基础上添加新点，节省实验成本
