# 第1章 Introduction — 实验设计导论

## 📌 本章概要

介绍实验设计 (Design of Experiments, DoE) 的四大目标和基本概念。通过对比线性回归和高斯过程回归，展示为什么需要精心选择实验点。

---

## 1.1 实验设计的四大目标

### 🎯 目标 1：逼近粗糙函数 (Approximate a Rough Function)

粗糙函数指变化剧烈、具有多峰值的函数。例如：

$$f(x) = \frac{\sin(10\pi x)}{1 + 64(x-0.5)^2}$$

**设计策略**：在变化剧烈的地方放置更多实验点。均匀分布的点对于这种函数效果较差。

### 🎯 目标 2：逼近光滑函数 (Approximate a Smooth Function)

光滑函数相对平缓。例如：

$$f(x) = \frac{1}{1 + 64(x-0.5)^2}$$

**设计策略**：可以在整个区域均匀布点，或者根据函数的曲率分布点。

### 🎯 目标 3：优化 (Optimization)

目标是找到使 $f(x)$ 最大（或最小）的点。例如双峰混合分布：

$$f(x) = \frac{2}{3}\text{Beta}(x|2,10) + \frac{1}{3}\text{Beta}(x|10,2)$$

**设计策略**：在可能有最优值的位置（如峰值附近）多布点。

### 🎯 目标 4：不确定性传播 (Uncertainty Propagation)

输入 $X$ 服从某个概率分布，目标是了解输出 $Y = f(X)$ 的分布。设计点应从输入分布中采样。

## 1.2 线性回归 vs 高斯过程回归

### 示例函数

$$f(x) = \frac{\sin(10\pi x)}{1 + 64(x-0.25)^2} + x^2$$

### 线性回归 (Linear Regression)

- 假设函数形式为 $y = \beta_0 + \beta_1 x + \varepsilon$
- 使用最小二乘法估计参数
- 预测区间宽度由假设的模型形式决定

**预测区间计算**（假设 $y = X\beta + \varepsilon, \varepsilon \sim N(0,\sigma^2)$）:

$$\hat{y}(x_{\text{new}}) = x_{\text{new}}^T \hat{\beta}$$

$$ \text{Var}[\hat{y}(x_{\text{new}})] = \sigma^2 \left[1 + x_{\text{new}}^T (X^T X)^{-1} x_{\text{new}}\right] $$

95% 预测区间: $\hat{y} \pm t_{0.975, n-p} \cdot \text{se}(\hat{y})$

### 高斯过程回归 (Gaussian Process Regression)

- 不假设具体的函数形式
- 假设函数值服从多元正态分布
- 使用核函数（如高斯核）度量点之间的相关性
- 预测时提供均值和方差

**高斯核**：$k(x_i, x_j) = \tau^2 \exp\left[-\left(\frac{|x_i-x_j|}{\theta}\right)^2\right]$

其中 $\tau^2$ 是过程方差，$\theta$ 是相关长度参数。

**GP预测均值**：$\hat{y}(x^*) = \mu + r(x^*)^T R^{-1}(y - \mu\mathbf{1})$

**GP预测方差**：$s^2(x^*) = \tau^2\left[1 - r(x^*)^T R^{-1} r(x^*)\right]$

## 📝 例题详解

### 例题 1.1：设计点数量与逼近精度

**问题**：对于函数 $f(x) = \sin(10\pi x) / (1 + 64(x-0.5)^2)$，如果用 $n=10$ 个等间距点进行插值，最大误差大约是多少？如果增加到 $n=20$ 呢？

**解**：
等间距点：$D_i = (i-0.5)/n, i=1,\dots,n$

对于 $n=10$，点位于 0.05, 0.15, ..., 0.95。在 $x=0.5$ 附近函数变化剧烈，等间距点无法捕捉到峰值，最大误差较大。

对于 $n=20$，点更密集，误差会减小，但等间距在变化剧烈区域仍然不够。

**关键洞察**：这不是点数量的问题，而是点分布的问题。应该在不光滑的区域多布点。

### 例题 1.2：Beta 分布分位数做设计点

**问题**：对于光滑函数 $f(x) = 1/(1+64(x-0.5)^2)$，解释为什么用 $\text{Beta}(0.5, 0.5)$ 的分位数作为设计点。

**解**：
- $\text{Beta}(0.5, 0.5)$ 的概率密度函数在 $x=0$ 和 $x=1$ 处趋于无穷大，在 $x=0.5$ 处最小
- 函数 $f(x)$ 在 $x=0$ 和 $x=1$ 附近变化较快
- 因此 Beta(0.5,0.5) 的分位数会在两端更密集，正好匹配函数变化剧烈的区域

```python

# D 的值在 0 和 1 附近更密集
```

## 💡 核心要点

1. **没有万能的设计**：不同的目标（逼近、优化、不确定性传播）需要不同的设计策略
2. **线性回归的局限**：假设函数形式已知，否则预测不可靠
3. **GP 的优势**：灵活的非参数模型，能自动适应数据，并提供不确定性量化
4. **R 辅助策略**：PyED4DSE 项目使用 Python 做可视化，用 R 做专业计算（如 rkriging 包）
