## 5.1 均匀性 (Uniformity)

### 一维均匀性

用 $n$ 个点 $d_1, \dots, d_n$ 代表 $[0,1]$ 上的均匀分布。

**经验分布函数** (ECDF)：
$$F_n(x) = \frac{1}{n}\sum_{i=1}^n I(d_i \leq x)$$

**最优设计**：使 ECDF 最接近理论 CDF $F(x)=x$。

$$D_i = \frac{i - 0.5}{n}, \quad i = 1,\dots,n$$

```python
# 随机设计
x_rand = np.sort(rng.random(n))
# 经验 CDF 是阶梯函数
ax.step([0] + list(x_rand) + [1], 
        [0] + list(np.arange(1, n+1)/n) + [1],
        where="post")

# 最优设计
x_opt = (np.arange(1, n+1) - 0.5) / n
```

### 二维均匀性 — 差异度 (Discrepancy)

差异度衡量设计点与均匀分布的偏离程度：

$$D_n = \sup_{x \in [0,1]^p} \left|\frac{1}{n}\sum_{i=1}^n I(d_i \leq x) - \prod_{j=1}^p x_j\right|$$

这就是 **Kolmogorov-Smirnov 统计量**的多元推广。

## 5.3 变换方法 (Transformation Methods)

### 逆变换法

如果 $D$ 在 $[0,1]^p$ 上均匀分布，则对任意目标分布 $F$：

$$X = F^{-1}(D) \quad \text{按元素应用}$$

使 $X$ 服从 $F$ 分布。

### 正态分布

$$X_i = \Phi^{-1}(D_i), \quad i=1,\dots,p$$

其中 $\Phi^{-1}$ 是标准正态逆 CDF。

```python
from scipy.stats import norm
D_norm = norm.ppf(d)  # d 是 [0,1]^2 上的均匀设计
```

### Gamma 分布

$$X_i = G^{-1}(D_i; \alpha, \beta), \quad i=1,\dots,p$$

```python
from scipy.stats import gamma as sp_gamma
D_gamma = sp_gamma.ppf(d, 2, scale=1)  # Gamma(2, 1)
```

## 5.4 Rosenblatt 变换

### 相关正态分布

对于相关结构，使用 Rosenblatt 变换（条件分布方法）：

对于二元正态 $N(0, \Sigma)$，其中 $\Sigma = \begin{bmatrix}1 & \rho \\ \rho & 1\end{bmatrix}$：

$$x_1 = \Phi^{-1}(d_1)$$
$$x_2 = \Phi^{-1}(d_2; \mu=\rho x_1, \sigma^2=1-\rho^2)$$

```python
rho = 0.5
x1 = norm.ppf(d[:, 0])
x2 = norm.ppf(d[:, 1], loc=rho * x1, scale=np.sqrt(1 - rho**2))
```

## 5.5 支持点 (Support Points)

### 定义

支持点是使**能量距离**最小化的代表点：

$$S_n = \arg\min_{S \subset \mathcal{X}, |S|=n} \mathcal{E}(S, P)$$

其中能量距离衡量 $S$ 的经验分布与目标分布 $P$ 之间的差异：

$$\mathcal{E}(S, P) = \frac{2}{n}\sum_{i=1}^n \mathbb{E}_P[\|s_i - X\|] - \frac{1}{n^2}\sum_{i=1}^n\sum_{j=1}^n \|s_i - s_j\| - \mathbb{E}_P[\|X - X'\|]$$

### 支持点 vs 变换方法

| 方面 | 变换方法 | 支持点 |
|:----:|:--------:|:------:|
| 适用范围 | 已知逆 CDF 的分布 | 仅需样本的任意分布 |
| 计算复杂度 | 低 | 较高（需要优化） |
| 高维性能 | 可扩展到高维 | 高维困难 |
| 相关性 | 需显式指定相关结构 | 自动匹配样本相关性 |

### 示例：支持点生成

```python
# 通过 R 的 support 包生成支持点
out = run_wrapper(
    "ch5_support_points.R",
    inputs={"params.csv": pd.DataFrame({
        "n": [50], "p": [2], 
        "dist_type": ["normal"], "seed": [1]
    })},
    required_packages=["support", "mnormt"],
)
D = out["support_points.csv"].to_numpy()
```

### 高级支持点

- **相关正态分布** ($\rho=0.5$)：支持点自然保持相关性
- **香蕉形密度**：支持点能适应复杂分布形状

## 5.6 不确定性传播 (Borehole 函数)

### Borehole 函数

$$f(x) = \frac{2\pi T_u(H_u - H_l)}{\ln(r/r_w)\left[1 + \frac{2LT_u}{\ln(r/r_w)r_w^2 K_w} + \frac{T_u}{T_l}\right]}$$

8 个输入变量：
- $r_w \in [0.05, 0.15]$（钻孔半径）
- $r \in [100, 50000]$（影响半径）
- $T_u \in [63070, 115600]$（上含水层导水系数）
- $H_u \in [990, 1110]$（上含水层水头）
- $T_l \in [63.1, 116]$（下含水层导水系数）
- $H_l \in [700, 820]$（下含水层水头）
- $L \in [1120, 1680]$（钻孔长度）
- $K_w \in [9855, 12045]$（钻孔水力传导率）

### 不确定性传播流程

1. 定义输入的联合分布（$X_1$ 正态，$X_2$ 对数正态，其余均匀）
2. 用支持点或 MC 采样 $n=100$ 个代表点
3. 计算每个代表点的 Borehole 函数值
4. 用 KDE 估计输出分布

```python
# MC: n=100 随机样本
idx_mc = rng.choice(N, n, replace=False)
mc_y = true_y[idx_mc]

# 支持点: 用 R 的 support 包
out = run_wrapper("ch5_support_points.R", ...)
D_sp = out["support_points.csv"].to_numpy()
sp_y = np.array([borehole(x) for x in D_sp_scaled])
```

**结果**：支持点比 MC 更准确地恢复了真实输出分布。

## 📝 例题详解

### 例题 5.1：差异度与积分精度的关系

**问题**：为什么差异度小的设计在蒙特卡洛积分中表现更好？

**解**：
Koksma-Hlawka 不等式给出了数值积分误差的上界：

$$\left|\frac{1}{n}\sum_{i=1}^n f(d_i) - \int_{[0,1]^p} f(x)dx\right| \leq V(f) \cdot D_n$$

其中 $V(f)$ 是 $f$ 的 Hardy-Krause 变差，$D_n$ 是差异度。因此差异度直接控制积分误差。

### 例题 5.2：Sobol 序列的收敛速度

**问题**：MC 的收敛速度是 $O(n^{-1/2})$，而 Sobol 序列是 $O((\log n)^p/n)$。对于 $p=2$，当 $n$ 多大时 Sobol 开始优于 MC？

**解**：
$n^{-1/2} = (\log n)^2/n$ 时，解得大约 $n \approx 100$。所以对于中大规模问题，Sobol 显著优于 MC。

### 例题 5.3：支持点的计算效率

**问题**：支持点需要优化求解，为什么值得用？

**解**：
1. 对于 Borehole 函数（1次计算约 1ms），$n=100$ 支持点只需要 100 次函数评估
2. MC 需要 $N=100000$ 才能收敛到相同精度
3. 当函数评估昂贵（如 CFD 模拟）时，100 vs 100000 是巨大差异

## 💡 核心要点

1. **差异度**衡量设计的均匀性，与蒙特卡洛积分误差直接相关
2. **Sobol 序列**和**均匀设计**的收敛速度远快于纯随机的 MC 方法
3. **逆变换法**将均匀设计转换为任意目标分布的代表点
4. **Rosenblatt 变换**处理相关结构
5. **支持点**通过能量距离优化，能适应任意复杂分布
6. **不确定性传播**中，少量精心选择的代表点（如支持点）可以替代大量 MC 样本，显著降低计算成本
