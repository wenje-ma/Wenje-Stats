## 11.1 因子选择问题

### 问题定义

有 $p$ 个输入变量（因子），$n$ 个观测数据 $(x_i, y_i)$。哪些因子对响应 $y$ 影响最大？

### 特殊挑战：相关性

当因子间存在相关性时，传统分析方法（如 Sobol' 指标、主效应图）可能失效。例如：

$$y = \sqrt{2}x_1 + \frac{2}{\sqrt{3}}x_2 + x_3 + \varepsilon$$

当 $x_1$ 和 $x_2$ 相关（$\rho \neq 0$）时，方差分解不唯一。

---

## 11.2 效应与总 Sobol' 方差

### 独立因子情况

当因子独立时，效应和总 Sobol' 方差有简单的解析形式。

对于线性模型 $y = \beta_1 x_1 + \beta_2 x_2 + \beta_3 x_3$（$x_i \in [0,1]$ 独立）：

**效应**：$f_i(x_i) = \beta_i(x_i - 0.5)$

```python
# 理论效应曲线
x = np.linspace(0, 1, 100)
ax.plot(x, np.sqrt(2) * x, label=r"$x_1$")   # beta1 = sqrt(2)
ax.plot(x, 2/np.sqrt(3) * x, label=r"$x_2$") # beta2 = 2/sqrt(3)
ax.plot(x, 1 * x, label=r"$x_3$")            # beta3 = 1
```

**总 Sobol' 方差**（对于 $y = \beta_1 x_1 + \beta_2 x_2 + \beta_3 x_3$，$x_i$ 独立）：

$$V_i^{\text{tot}} = \beta_i^2 \cdot \text{Var}(x_i) = \beta_i^2 / 12$$

当 $x_1$ 和 $x_2$ 相关时：

$$V_1^{\text{tot}} = \beta_1^2 \text{Var}(x_1) + \beta_1\beta_2 \text{Cov}(x_1, x_2)$$

$$V_2^{\text{tot}} = \beta_2^2 \text{Var}(x_2) + \beta_1\beta_2 \text{Cov}(x_1, x_2)$$

```python
# 相关情况下的总 Sobol' 方差
rho_vals = np.linspace(-1, 1, 100)
ax.plot(rho_vals, 2 * (1 - rho_vals**2), label=r"$x_1$")   # V1_tot
ax.plot(rho_vals, 4/3 * (1 - rho_vals**2), label=r"$x_2$") # V2_tot
ax.axhline(y=1, label=r"$x_3$")                             # V3_tot
```

## 11.3 FIRST 算法

### 背景

FIRST (Factor Importance Ranking and Selection using Target) 算法专门处理**相关输入**下的因子选择。

### 思路

1. 拟合函数 $f(x) = \mathbb{E}[Y|X=x]$
2. 对目标（响应）拟合一个替代模型
3. 通过替换（permutation）或条件方差衡量因子重要性

### 示例：相关变量 ($\rho=0.9$)

模型：$y = \sqrt{2}x_1 + \frac{2}{\sqrt{3}}x_2 + x_3 + \varepsilon$

当 $x_1$ 和 $x_2$ 高度相关时：

```python
# FIRST 算法结果
out = run_wrapper("ch11_first_example.R",
    inputs={"params.csv": pd.DataFrame({
        "seed": [123], "N": [1000], "rho": [0.9]
    })},
    required_packages=["MASS", "first"],
)
first_scores = out["first_scores.csv"]
```

**结果**：
- $x_1$ 和 $x_2$ 高度相关，FIRST 对两者的重要性贡献相仿（因为无法区分两者的独特贡献）
- $x_3$ 独立，其重要性可以单独识别
- 去除 $x_2$ 后，$x_1$ 的重要性大幅上升（因为 $x_1$ 吸收了本该属于 $x_2$ 的贡献）

## 11.5 TwinGP（双高斯过程）

### 动机

当数据有复杂的非线性关系时，标准 GP 可能无法充分利用数据中的结构信息。

### TwinGP 方法

TwinGP 使用两个 GP：一个建模全局趋势，另一个建模局部细节。

$$y = f_{\text{global}}(x) + f_{\text{local}}(x) + \varepsilon$$

### 风能数据应用

```python
out = run_wrapper("ch11_twingp.R",
    inputs={
        "params.csv": pd.DataFrame({"seed": [123]}),
        "wind.csv": wind,
    },
    required_packages=["twingp"],
)
pred = out["prediction.csv"]
```

**可视化**：
- 灰色 'x'：原始数据
- 蓝色曲线：TwinGP 拟合

TwinGP 在风速-功率曲线中表现优异，尤其是在功率曲线的陡峭上升段和饱和段。

## 📝 例题详解

### 例题 11.1：相关变量下的方差分解

**问题**：当因子 $x_1$ 和 $x_2$ 相关时，为什么 Sobol' 分解不再唯一？

**解**：
对于 $y = \beta_1 x_1 + \beta_2 x_2$，总方差：

$$V = \beta_1^2 V_1 + \beta_2^2 V_2 + 2\beta_1\beta_2\text{Cov}(x_1, x_2)$$

方差可以分解为：

$$V_1 = \beta_1^2 V_1 + \beta_1\beta_2\text{Cov}(x_1, x_2)$$
$$V_2 = \beta_2^2 V_2 + \beta_1\beta_2\text{Cov}(x_1, x_2)$$

但这把协方差项分配给了两个变量，这种分配方式不唯一（取决于 ANOVA 分解中项的排序）。

### 例题 11.2：FIRST 的优势

**问题**：相比传统的 Sobol' 指标，FIRST 在相关变量下有什么优势？

**解**：
- Sobol' 指标假设变量独立，在相关时给出有偏的重要性估计
- FIRST 通过条件方差计算，不受变量顺序影响
- FIRST 能识别在给定其他变量后，某个变量的**增量**贡献
- 这对于筛选实验设计特别有价值——设计者想知道"如果我不测量 $x_i$，会损失多少信息"

### 例题 11.3：RF 和 GP 的比较

**问题**：RF 和 GP 在因子选择中各有何优缺点？

**解**：

| 方面 | 随机森林 (RF) | 高斯过程 (GP) |
|:----:|:------------:|:-------------:|
| 非线性 | 自然处理 | 通过核函数处理 |
| 可解释性 | 变量重要性易得 | 需额外计算 Sobol' |
| 外推能力 | 有限（只能内插） | 可外推但需谨慎 |
| 计算成本 | 低 | 高 ($O(n^3)$) |
| 变量选择 | 内置重要性指标 | 需外部方法 (FIRST) |
| 不确定性 | 无直接量化 | 自然提供 |

### 练习 11.6：理论题
证明：对于线性模型 $y = X\beta + \varepsilon$，其中 $X \sim N(0, \Sigma)$：

$$S_i = \frac{\beta_i^2 \sigma_i^2 + \sum_{j \neq i} \beta_i \beta_j \sigma_{ij}}{\beta^T \Sigma \beta + \sigma_\varepsilon^2}$$

其中 $\sigma_{ij} = \text{Cov}(x_i, x_j)$，$\sigma_i^2 = \text{Var}(x_i)$。

## 💡 核心要点

1. **因子相关性**使传统的方差分解失效，需要特殊方法（如 FIRST）
2. **FIRST 算法**通过条件方差计算，适用于相关输入
3. **RF 和 GP**都能进行因子选择，各有优缺点
4. **因子选择**可以显著降低实验成本——只关注重要变量
5. **TwinGP** 通过双 GP 结构提高复杂函数的拟合精度
6. 数据分析应该是迭代的：先筛选重要因子，再精细建模
