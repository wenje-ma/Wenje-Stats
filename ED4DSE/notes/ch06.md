## 6.1 敏感性分析概述

**目标**：确定哪些输入变量对输出变异的贡献最大。

**两类方法**：
- **基于方差** (Variance-based)：通过方差分解量化每个变量的贡献
- **基于导数** (Derivative-based)：通过函数梯度衡量局部敏感性

---

## 6.2 Sobol' 敏感性分析

### 方差分解 (ANOVA 分解)

假设 $f(x) = f(x_1, \dots, x_p)$ 定义在 $[0,1]^p$ 上：

$$f(x) = f_0 + \sum_i f_i(x_i) + \sum_{i<j} f_{ij}(x_i, x_j) + \cdots$$

其中各成分正交且均值为零。总方差分解为：

$$V = \sum_i V_i + \sum_{i<j} V_{ij} + \cdots$$

### Sobol' 指标

**一阶 Sobol' 指标**（主效应）：

$$S_i = \frac{V_i}{V} = \frac{\text{Var}_{x_i}[\mathbb{E}(Y|x_i)]}{\text{Var}(Y)}$$

**总 Sobol' 指标**（主效应 + 所有交互作用）：

$$T_i = \frac{\mathbb{E}_{x_{-i}}[\text{Var}(Y|x_{-i})]}{\text{Var}(Y)} = 1 - \frac{\text{Var}_{x_{-i}}[\mathbb{E}(Y|x_{-i})]}{\text{Var}(Y)}$$

其中 $x_{-i}$ 表示除 $x_i$ 外的所有变量。

**性质**：
- $0 \leq S_i \leq T_i \leq 1$
- $S_i$ 小但 $T_i$ 大 → $x_i$ 主要通过交互作用影响输出
- $\sum_i S_i \leq 1$，$\sum_i T_i \geq 1$

### 主效应图 (Main Effects)

主效应函数：$f_i(x_i) = \mathbb{E}_{x_{-i}}[Y|x_i] - f_0$

绘制当其他变量在平均水平时，输出随单个变量的变化。

## 6.3 DELSA (Derivative-based Local Sensitivity Analysis)

### 导数敏感性指标

对于平方可积函数，定义：

$$\nu_i = \mathbb{E}\left[\left(\frac{\partial f}{\partial x_i}\right)^2\right]$$

对 Borehole 函数，$\nu_i$ 衡量 $x_i$ 对输出的平均局部敏感性。

### 与 Sobol' 的关系

对于定义在 $[0,1]^p$ 上的函数：

$$\nu_i \cdot \sigma_i^2 \geq S_i$$

其中 $\sigma_i^2$ 是 $x_i$ 的方差。这个不等式说明导数敏感性和方差敏感性是相关的。

## 6.4 方差 vs 导数敏感性对比

测试函数（$p=4$）：

$$f(x) = \sum_{k=1}^p \frac{\sin((k^2+1)\pi x_k)}{k}$$

**特点**：
- $x_1$ 振幅最大（$k=1$, $\sin(2\pi x)$）
- $x_4$ 振幅最小（$k=4$, $\sin(17\pi x)/4$）
- 但 $x_4$ 的频率最高，导数最大

**对比结果**：
- **Sobol' 指标** $T_i$：$x_1 > x_2 > x_3 > x_4$（基于方差）
- **DELSA 指标** $\bar{\nu}_i$：$x_4 > x_3 > x_2 > x_1$（基于导数）

**结论**：两种方法给出不同排名，各有利弊。

## 6.5 Morris 筛选设计

### 基本思想

在 $[0,1]^p$ 上，将每个变量离散化为 $l$ 个水平（通常 $l=4$ 或 $l=6$）。

**基本效应** (Elementary Effect)：

$$EE_i(x) = \frac{f(x_1, \dots, x_i + \Delta, \dots, x_p) - f(x)}{\Delta}$$

其中 $\Delta = \frac{l}{2(l-1)}$。

### Morris 指标

- $\mu_i$：$EE_i$ 的均值（衡量整体影响）
- $\mu_i^*$：$|EE_i|$ 的均值（防止正负抵消）
- $\sigma_i$：$EE_i$ 的标准差（衡量非线性/交互作用）

### 严格 OFAT vs 标准 OFAT

**严格 OFAT**：每次只改变一个变量，在改变 $x_1$ 的基础上再改变 $x_2$。
**标准 OFAT**：每次从基础点（A）出发，分别改变 $x_1$ 和 $x_2$。

```python
# 标准 OFAT (p=2, l=4)
delta = 1/3
for i in range(4):
    C1[i, 0] = A[i, 0] + delta  # 改变 x1
    C2[i, 1] = A[i, 1] + delta  # 从 A 改变 x2
```

## 6.6 Sobol' 设计

Sobol' 敏感性分析使用特殊的两矩阵设计：

- $A$: $n \times p$ 矩阵
- $B$: 另一个独立的 $n \times p$ 矩阵
- $C_i$: 用 $B$ 的第 $i$ 列替换 $A$ 的第 $i$ 列

$$f(A) \text{ 和 } f(C_i) \text{ 的差异用于估计 } S_i$$

## 6.7 MOFAT 设计

### 原理

MOFAT (Multiple-Output Factor Analysis Technique) 是 Morris 方法的推广，允许同时处理多个输出。

**设计结构**：
- $l$ 个基础点（A 矩阵）
- $l$ 个只改变 $x_1$ 的点（C1 矩阵）
- $l$ 个只改变 $x_2$ 的点（C2 矩阵）

**投影 MOFAT**：在 MOFAT 基础上优化投影性质。

### 例题 6.1：Borehole 函数的 Sobol' 分析

**问题**：基于 Sobol' 分析结果，Borehole 函数中哪些变量最重要？

**解**：
在典型参数设置下：
- $x_3$ ($T_u$), $x_4$ ($H_u$), $x_6$ ($H_l$) 通常最重要（一阶 + 总效应都大）
- $x_8$ ($K_w$) 主要通过交互作用影响（一阶小，总效应大）
- $x_1$ ($r_w$) 和 $x_7$ ($L$) 相对不重要

### 例题 6.2：Morris 方法中 $\mu$ 和 $\sigma$ 的解释

**问题**：如何理解 Morris 的 $(\mu, \sigma)$ 图？

**解**：
- $\mu_i$ 大 → 变量 $x_i$ 对输出影响大
- $\sigma_i$ 大 → $x_i$ 的效应是非线性的或涉及交互作用
- $\mu_i$ 小但 $\sigma_i$ 大 → 效应正负抵消（需要看 $\mu_i^*$）
- 对角线 $(\mu = \sigma)$ 以下：主要是线性主效应
- 对角线以上：存在显著的非线性/交互作用

## 💡 核心要点

1. **Sobol' 指标**基于方差分解，提供详细的敏感性分析但计算成本高
2. **DELSA** 基于导数，计算成本低，可能对高频变量更敏感
3. **Morris 方法**在高维问题时经济高效（约 $r(p+1)$ 次函数评估）
4. **MOFAT** 是 Morris 的扩展，同时对多个输出进行分析
5. 筛选设计的核心目标是在少量实验下识别重要变量
6. 不同方法可能给出不同排名，理解各方法的假设和局限很重要
