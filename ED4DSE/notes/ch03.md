# 第3章 Model-based Designs — 基于模型的设计

## 📌 本章概要

在假设 $f$ 来自高斯过程的前提下，如何选择最优的实验点。核心准则包括 IMSE (积分均方误差)、MMSE (最大均方误差) 和最大熵。**本章全为纯 Python 实现，无 R 依赖。**

---

## 3.1 模型假设

假设真实函数 $f$ 来自一个已知协方差结构的高斯过程：

$$f \sim \mathcal{GP}(0, \tau^2 R(\cdot, \cdot))$$

使用高斯核：$R(x_i, x_j) = \exp\left[-\left(\frac{|x_i-x_j|}{\theta}\right)^2\right]$

在给定设计点 $D$ 和观测值 $y$ 后，在测试点 $x^*$ 处的预测方差为：

$$s^2(x^*|D) = \tau^2 \left[1 - r(x^*)^T R^{-1} r(x^*)\right]$$

其中 $r_i(x^*) = \exp\left[-\left(\frac{|x^*-D_i|}{\theta}\right)^2\right]$，$R_{ij} = \exp\left[-\left(\frac{|D_i-D_j|}{\theta}\right)^2\right]$

## 3.2 随机设计

最简单的设计方法：在输入空间随机均匀布点。

```python
p = 2  # 维度
n = 7  # 点数
rng = np.random.default_rng(seed)
D = rng.random((n, p))
```

**缺点**：随机点可能聚集或留下大的空隙。

## 3.3 RMSE 曲线 (Root Mean Squared Error)

RMSE 在 $x$ 处的定义为：

$$\text{RMSE}(x|D) = \sqrt{s^2(x|D)} = \tau\sqrt{1 - r(x)^T R^{-1} r(x)}$$

```python
def rmse_curve(D, theta, test):
    """计算测试网格上各点的 RMSE"""
    n = D.size
    E = np.abs(D[:, None] - D[None, :])
    R = kernel_rbf(E, theta)
    L = cholesky(R + 1e-6 * np.eye(n), lower=False)
    
    def s(x):
        r = np.exp(-(((x - D) / theta) ** 2))
        a = solve_triangular(L.T, r, lower=True)
        v = 1.0 - float(np.sum(a * a))
        return float(np.sqrt(max(v, 0.0)))
    
    return np.array([s(float(t)) for t in test])
```

---

## 3.4 IMSE — 积分均方误差

IMSE 在整个设计空间上积分预测方差：

$$\text{IMSE}(D) = \int_0^1 s^2(x|D) dx$$

由于 $s^2(x|D) = \tau^2 \left[1 - r(x)^T R^{-1} r(x)\right]$，且 $\tau^2$ 是常数，我们最小化：

$$\int_0^1 \left[1 - r(x)^T R^{-1} r(x)\right] dx = 1 - \text{tr}\left(R^{-1} \int_0^1 r(x)r(x)^T dx\right)$$

```python
def imse(D, theta):
    """计算 IMSE"""
    n = D.size
    E = np.abs(D[:, None] - D[None, :])
    R = kernel_rbf(E, theta)
    L = cholesky(R + 1e-6 * np.eye(n), lower=False)
    
    def s2(x):
        r = np.exp(-(((x - D) / theta) ** 2))
        a = solve_triangular(L.T, r, lower=True)
        v = 1.0 - float(np.sum(a * a))
        return float(max(v, 0.0))
    
    grid = np.linspace(0.0, 1.0, 2001)
    vals = np.array([s2(float(g)) for g in grid])
    return float(trapezoid(vals, grid))
```

---

## 3.5 MMSE — 最大均方误差

MMSE 关注最坏情况：

$$\text{MMSE}(D) = \max_{x \in [0,1]} s^2(x|D)$$

IMSE-optimal 和 MMSE-optimal 设计通常不同——前者关注平均表现，后者关注最坏情况。

```python
def mmse(D, theta):
    """计算 MMSE"""
    test = np.linspace(0.0, 1.0, 1001)
    sa = rmse_curve(D, theta, test)
    return float(np.max(sa * sa))
```

---

## 3.6 IMSE-optimal 和 MMSE-optimal 设计

通过数值优化寻找最优设计点：

```python
def optimize_design(D0, theta, objective):
    """优化设计点"""
    bounds = [(0.0, 1.0)] * D0.size
    
    if objective == "imse":
        f = lambda x: imse(x, theta)
    elif objective == "mmse":
        f = lambda x: mmse(x, theta)
    
    res = minimize(f, D0, method="L-BFGS-B", bounds=bounds)
    return np.asarray(res.x, float)
```

### 四种设计对比

对于 $\theta=0.1, n=10$：

| 设计类型 | 特点 | IMSE | MMSE |
|:--------:|:----:|:----:|:----:|
| 等间距 | 简单均匀 | 较高 | 较高 |
| Chebyshev | 两端密集 | 中等 | 中等 |
| IMSE-optimal | 最小化平均方差 | 最低 | 中等 |
| MMSE-optimal | 最小化最大方差 | 中等 | 最低 |

## 3.7 相对效率 (Relative Efficiency)

相对效率衡量一个设计相对于最优设计的效率：

$$\text{RE}_{\text{IMSE}} = \frac{\text{IMSE}(D_{\text{min}})}{\text{IMSE}(D)}$$

$$\text{RE}_{\text{MMSE}} = \frac{\text{MMSE}(D_{\text{min}})}{\text{MMSE}(D)}$$

当 $\text{RE} = 1$ 时达到最优，$\text{RE} < 1$ 表示设计效率较低。

**关键发现**：在 $\theta$ 较小时，等间距设计的效率远低于最优设计；当 $\theta$ 较大时，所有设计趋于一致。

```python
# 计算相对效率 vs theta
ev = np.linspace(0.01, 0.4, 100)
IR = np.zeros((ev.size, 4))  # 4 种设计
MR = np.zeros((ev.size, 4))

for i, th in enumerate(ev):
    IR[i, :] = [imse(D1, th), imse(D2, th), imse(D3, th), imse(D4, th)]
    MR[i, :] = [mmse(D1, th), mmse(D2, th), mmse(D3, th), mmse(D4, th)]

# 相对效率
IRR = IR / IR.min(axis=1, keepdims=True)
MRR = MR / MR.min(axis=1, keepdims=True)
RE_IMSE = 1.0 / IRR
RE_MMSE = 1.0 / MRR
```

## 3.8 最大熵设计 (Maximum Entropy Design)

熵衡量不确定性。对于高斯过程，设计 $D$ 的熵（忽略常数）为：

$$H(D) = \frac{1}{2} \log \det(R(\theta))$$

**最大熵设计**选择最大化 $\det(R)$ 的设计点，这等价于最小化预测方差。

```python
def entropy(D, theta):
    """计算熵（行列式）"""
    E = np.abs(D[:, None] - D[None, :])
    R = kernel_rbf(E, theta)
    return float(np.linalg.det(R))
```

### 三种设计对比（$\theta=0.1, n=10$）

| 设计类型 | IMSE | MMSE | 熵 |
|:--------:|:----:|:----:|:-:|
| IMSE-optimal | 最低 | 中等 | - |
| MMSE-optimal | 中等 | 最低 | - |
| 最大熵 | 略高 | 略高 | 最大 |

**重要结论**：最大熵设计通常与 IMSE/MMSE 最优设计不同，但差异不大。

## 📝 例题详解

### 例题 3.1：Cholesky 分解加速

**问题**：在计算 RMSE 曲线时，为什么要用 Cholesky 分解而不是直接计算矩阵逆？

**解**：
对于每个测试点 $x^*$，需要计算 $r(x^*)^T R^{-1} r(x^*)$。直接求逆 $R^{-1}$ 的复杂度为 $O(n^3)$。用 Cholesky 分解 $R = LL^T$，则：

$$r^T R^{-1} r = r^T L^{-T} L^{-1} r = (L^{-1}r)^T (L^{-1}r) = a^T a$$

只需求解一次三角系统 $La = r$，复杂度 $O(n^2)$。对整个测试网格的累计加速显著。

### 例题 3.2：IMSE 的解析推导

**问题**：证明 $\text{IMSE}(D) = 1 - \text{tr}\left(R^{-1} \int_0^1 r(x)r(x)^T dx\right)$（假设 $\tau^2=1$）。

**解**：

$$s^2(x) = 1 - r(x)^T R^{-1} r(x)$$

$$\text{IMSE} = \int_0^1 [1 - r(x)^T R^{-1} r(x)] dx = 1 - \int_0^1 r(x)^T R^{-1} r(x) dx$$

由于 $r(x)^T R^{-1} r(x) = \text{tr}(R^{-1} r(x) r(x)^T)$（标量的迹等于自身），所以：

$$\int_0^1 r(x)^T R^{-1} r(x) dx = \text{tr}\left(R^{-1} \int_0^1 r(x)r(x)^T dx\right)$$

因此 $\text{IMSE} = 1 - \text{tr}\left(R^{-1} A\right)$，其中 $A_{ij} = \int_0^1 r_i(x)r_j(x) dx$。

### 例题 3.3：数值积分精度

**问题**：为什么 IMSE 计算中使用 2001 个网格点？这对结果的影响是什么？

**解**：
使用梯形法则近似积分。2001 个点意味着步长为 $h = 1/2000 = 0.0005$，对大多数 $\theta$ 值来说足够精确。

更精确的方法是用高斯积分或解析公式，但对设计优化来说，梯形法则的近似精度已经足够。

## 💡 核心要点

1. **IMSE 和 MMSE** 是两个最常用的基于模型的设计准则
2. **IMSE-optimal 设计** 在平均意义下最优，**MMSE-optimal 设计** 在最坏情况意义下最优
3. **数值优化**（L-BFGS-B）是寻找最优设计的关键工具
4. **最大熵设计** 提供了另一种视角，实际效果与 IMSE 设计相近
5. **$\theta$ 的影响**：$\theta$ 越小（函数越粗糙），设计优化越重要
6. **本章纯 Python**，无需 R 依赖
