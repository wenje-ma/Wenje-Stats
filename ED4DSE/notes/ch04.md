## 4.1 空间填充设计概述

空间填充设计的核心思想是：**在输入空间内均匀地散布实验点**，以便在没有任何函数先验的情况下，尽可能捕捉到整个空间的响应行为。

与第3章不同，空间填充设计**不依赖于模型假设**，因此更稳健。

## 4.2 聚类设计 (Clustering-based Design)

### 方法

1. 生成密集候选点集（如 $100\times100$ 网格）
2. 用 K-means 聚类将候选点聚为 $n$ 类
3. 以聚类中心作为设计点

```python
def kmeans_pp(X, k, seed=2, iters=50):
    """K-means++ 初始化聚类"""
    rng = np.random.default_rng(seed)
    idx = rng.choice(len(X), size=k, replace=False)
    C = X[idx].copy()
    
    for _ in range(iters):
        d = ((X[:, None, :] - C[None, :, :]) ** 2).sum(axis=2)
        lab = d.argmin(axis=1)
        C_new = np.vstack([X[lab == j].mean(axis=0) for j in range(k)])
        if np.allclose(C_new, C):
            break
        C = C_new
    return C

# 生成 100x100 网格候选点
def lattice(n, p):
    grid_1d = (np.arange(n) + 0.5) / n
    X, Y = np.meshgrid(grid_1d, grid_1d, indexing="xy")
    return np.column_stack([X.ravel(), Y.ravel()])

X = lattice(100, 2)  # 10000 个候选点
D = kmeans_pp(X, k=7, seed=2)
```

**优点**：设计点自然地在空间均匀散布  
**可视化**：Voronoi 图展示每个点的领域

## 4.3 Minimax 设计

### Minimax 准则

最小化从设计空间内任意点到最近设计点的最大距离：

$$\min_D \max_{x \in \mathcal{X}} \min_{d \in D} \|x - d\|$$

### 理论 Minimax 设计 $(n=7, p=2)$

对于 $[0,1]^2$ 上的 Minimax 设计（$n=7$），最优解的一部分已知：

## 4.4 Maximin 设计

### Maximin 准则

最大化设计点之间的最小距离：

$$\max_D \min_{i \neq j} \|d_i - d_j\|$$

### 图示

```python

```

## 4.5 拉丁超立方体设计 (LHD)

### 基本 LHD

将每个变量的 $[0,1]$ 区间分成 $n$ 等份，在每个区间内随机采样，保证每个变量的每个区间恰好有一个点。

```python
def lhs_random(n, p, seed=1):
    """随机拉丁超立方体"""
    rng = np.random.default_rng(seed)
    H = np.empty((n, p), float)
    for j in range(p):
        perm = rng.permutation(n)
        u = rng.random(n)
        H[:, j] = (perm + u) / n
    return H
```

### MmLHD (Maximin LHD)

在 LHD 基础上最大化最小距离准则。

**$n=4$ 示例**：
- 对角设计：$D = \{(0.125, 0.125), (0.375, 0.375), (0.625, 0.625), (0.875, 0.875)\}$
- MmLHD：点在对角线两侧散布，投影保持 Latin 性质

### Chebyshev-MmLHD

利用 Chebyshev 节点的思想，在 LHD 的边界处也放置点，提高边界预测精度。

## 4.6 最大投影设计 (MaxPro)

### MaxPro 准则

最大化所有设计点对之间距离的乘积：

$$\max_D \prod_{i < j} \|d_i - d_j\|$$

等价于最小化：

$$\min_D \sum_{i < j} \frac{1}{\|d_i - d_j\|^2}$$

### MaxPro 的投影性质

MaxPro 不仅在全空间追求均匀，而且在**每个低维投影子空间**也表现良好。这是它与简单 Maximin 的关键区别。

```python
# 计算各投影维度的最小距离
def min_mindist(M):
    """计算所有投影的最小距离"""
    n, p = M.shape
    out = np.zeros(p, float)
    for j in range(1, p):
        mins = []
        for cols in itertools.combinations(range(p), j):
            mins.append(float(np.min(pdist(M[:, cols]))))
        out[j-1] = float(np.min(mins))
    out[p-1] = float(np.min(pdist(M)))
    return out
```

## 4.8 验证实验 (Branin 函数)

Branin 函数是一个常用的测试函数：

```python
def branin(u):
    x1 = u[0] * 15 - 5
    x2 = u[1] * 15
    val1 = (x2 - 5.1/(4*np.pi**2)*(x1**2) + 5/np.pi*x1 - 6)**2
```

## 4.9 约束设计 (Constrained Design)

实际中，设计空间常有约束条件。例如：

$$c_1(x) = x_1 - \sqrt{50(x_2-0.52)^2 + 2} + 1 \leq 0$$
$$c_2(x) = \sqrt{120(x_2-0.48)^2 + 1} - 0.75 - x_1 \leq 0$$
$$c_3(x) = 0.65^2 - x_1^2 - x_2^2 \leq 0$$

### 方法

1. 生成大量 LHD 候选点
2. 筛选出满足约束的点
3. 使用 MaxProAugment 从可行候选点中选择设计

```python
# 生成候选点并筛选可行点
N = int(1e5)
lhs_all = lhs_random(N, 2, seed=1)
gval = np.apply_along_axis(constraint_vals, 1, lhs_all)
out_idx = (gval > 0).any(axis=1)
lhs_feasible = lhs_all[~out_idx]
```

## 4.13 贝叶斯计算设计

### 香蕉形分布 (Banana-shaped)

贝叶斯计算中，后验分布可能非常不规则。以香蕉形分布为例：

$$\log f(x_1, x_2) = -\frac{1}{2}\left(\frac{x_1^2}{100} + (x_2 + 0.03x_1^2 - 3)^2\right)$$

### MED (Minimum Energy Design)

最小能量设计用于在非均匀分布上放置代表性点。

**步骤**：
1. 用 MaxPro 生成初始设计
2. 通过 MCMC 采样目标分布
3. 用 MED 从 MCMC 样本中选择代表性点
4. 结合退火过程逐步优化

## 📝 例题详解

### 例题 4.1：Maximin vs Minimax

**问题**：解释 Maximin 和 Minimax 准则的区别，并举例说明何时使用哪个。

**解**：
- **Maximin**：最大化点间最小距离 → 关注"点与点的关系"
  - 适用于：希望设计点在空间均匀分散
  - 示例：$D_1 = \{(0,0), (1,1), (0,1), (1,0)\}$ 是 4 点的 Maximin 设计
- **Minimax**：最小化任意点到最近点的最大距离 → 关注"点与空间的关系"
  - 适用于：希望整个空间都被覆盖
  - 示例：4 点的 Minimax 设计是 $D_2 = \{(0.25,0.25), (0.25,0.75), (0.75,0.25), (0.75,0.75)\}$

### 例题 4.2：LHD 的投影性质

**问题**：为什么 LHD 的投影性质很重要？举例说明。

**解**：
在许多实际问题中，只有少数几个因子真正重要。如果一个设计在二维投影中聚集在一起，当其他因子被证明不重要时，有效设计点就减少了。

LHD 保证**每个变量的每个水平恰好有一个点**，因此在任何一维投影中都是均匀的。对比随机设计，LHD 在所有低维投影中都表现良好。

### 例题 4.3：MaxPro 的直觉

**问题**：解释 MaxPro 准则 $\sum_{i<j} 1/\|d_i-d_j\|^2$ 的几何含义。

**解**：
- 当两个设计点非常接近时，$1/\|d_i-d_j\|^2$ 很大 → 惩罚大
- 因此 MaxPro 倾向于让所有点对都互相远离
- 平方项意味着对非常接近的点对施加更重的惩罚
- 这与 Maximin 不同——Maximin 只关心最近的点对，而 MaxPro 考虑所有点对

## 💡 核心要点

1. **空间填充设计**不依赖模型假设，适用于探索性实验
2. **Maximin** 保证点间距离足够大；**Minimax** 保证整个空间被覆盖
3. **LHD** 保证一维投影的均匀性；**MmLHD** 结合了 LHD 和 Maximin
4. **MaxPro** 在高维投影中表现优异，是目前最先进的准则之一
5. **序贯设计**允许逐步添加实验点
6. **约束/定性/嵌套/分支**设计处理了各种实际问题中的特殊场景
7. **MED** 为贝叶斯计算中的非标准分布提供代表点
