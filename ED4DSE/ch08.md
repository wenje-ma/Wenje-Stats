## 8.1 两水平析因设计

### 示例：$2^{4-1}$ 部分因子设计

生成元：$D = ABC$（定义关系），因此 $ABCD = I$。

因子编码：$A, B, C = \pm 1$，$D = ABC$

```python
A = np.array([-1, -1, -1, -1, 1, 1, 1, 1])
B = np.array([-1, -1, 1, 1, -1, -1, 1, 1])
C = np.array([-1, 1, -1, 1, -1, 1, -1, 1])
D = A * B * C  # D = ABC
```

响应数据（化学实验产量）：
```python
y = np.array([504, 984, 928, 808, 992, 784, 464, 976])
```

### 半正态图 (Half-Normal Plot)

**步骤**：
1. 拟合线性模型 $y = X\beta + \varepsilon$，得到效应估计 $\hat{\beta}_j$
2. 计算效应的绝对值 $|\hat{\beta}_j|$
3. 排序后对半正态分位数作图

**半正态分位数**：

$$q_j = \Phi^{-1}\left(0.5 + 0.5 \cdot \frac{j-0.5}{m}\right), \quad j=1,\dots,m$$

其中 $m$ 是效应数量。

```python
from scipy.stats import norm

m = len(eff)
u = norm.ppf(0.5 + 0.5 * (np.arange(1, m+1) - 0.5) / m)
sorted_idx = np.argsort(np.abs(eff))

# 在图上标注效应名称
for i in range(len(u)):
    color = "red" if i >= m-3 else "blue"
    ax.text(u[i], sorted_abs_eff[i], eff_names[sorted_idx[i]], 
            color=color, ha="center", va="bottom")
```

**判断准则**：远离直线的点对应显著效应。

此例中，"2"（B因子）和其他几个效应显著。

## 8.2 贝叶斯半正态图 (HiGarrote)

### HiGarrote 方法

HiGarrote (Hierarchical Garrote) 是一种贝叶斯变量选择方法，通过收缩系数进行变量选择。

**与普通最小二乘 (OLS) 的对比**：

| 方法 | 系数 | 选择策略 |
|:----:|:----:|:--------:|
| OLS | 无偏 | 无 |
| HiGarrote | 收缩估计 | 自动选择 |
| 结果对比 | $R^2$ 略低，但 Adj $R^2$ 更高 | 模型更简洁 |

### Cast 疲劳数据分析

数据集：铝合金铸件疲劳实验

**全模型 vs HiGarrote 选择模型**：
- 全模型包含所有主效应和交互作用
- HiGarrote 筛选出关键变量
- 选择后的模型具有更高的调整 $R^2$

## 8.3 多水平设计

### 正交表 L18

L18 是 18 次运行的 Taguchi 正交表，支持 1 个两水平因子和 7 个三水平因子。

### MaxPro 准则 vs 能量距离

对 18 次运行的设计，从大量随机 L18 设计中选出最优：

- **MaxPro 准则**：$M(D) = \sum_{i<j} \frac{1}{\|d_i-d_j\|^p}$
- **能量距离**：$\text{ED}(D) = \frac{2}{n^2}\sum_{i<j} \|d_i-d_j\|$

## 8.4 混料设计 (Mixture Designs)

### 单纯形约束

混料设计中，各成分比例之和为 1：

$$x_1 + x_2 + \cdots + x_p = 1, \quad x_i \geq 0$$

### 三元坐标到笛卡尔坐标

```python
def ternary_to_cartesian(p1, p2, p3):
    """三元坐标 (p1, p2, p3) → 笛卡尔坐标 (x, y)
    p1 + p2 + p3 = 1
    """
    x = 0.5 * (2 * p2 + p3)
    y = (np.sqrt(3) / 2) * p3
    return x, y
```

### SLD (Simplex Lattice Design)

$\{x: x_1 + x_2 + x_3 = 1, x_i \geq 0\}$ 上的等间距网格。

例如 $\{0, 1/3, 2/3, 1\}$ 的格点设计（阶数 $m=3$）。

### SCD (Simplex Centroid Design)

顶点、边中点和重心组成的集合。

对于 $p=3$：顶点 (1,0,0)、(0,1,0)、(0,0,1)；边中点 (1/2,1/2,0) 等；重心 (1/3,1/3,1/3)。

### 空间填充混料设计

**无约束**：在整个单纯形上均匀布点。
**有约束**：满足 $l_i \leq x_i \leq u_i$ 的约束，在可行子区域内布点。

## 📝 例题详解

### 例题 8.1：$2^{4-1}$ 设计的别名结构

**问题**：$2^{4-1}$ 设计中，各效应的别名关系是什么？

**解**：
由于 $D = ABC$，定义关系为 $I = ABCD$。

各效应及其别名：
- $A = BCD$，$B = ACD$，$C = ABD$，$D = ABC$
- $AB = CD$，$AC = BD$，$AD = BC$

因此如果 $AB$ 显著，我们无法区分是 $AB$ 还是 $CD$ 的影响。

### 例题 8.2：半正态图的判断准则

**问题**：在半正态图上，为什么显著效应会偏离直线？

**解**：
- 不显著的效应估计值服从 $N(0, \sigma^2)$，其绝对值服从半正态分布
- 显著效应的均值非零，绝对值偏大
- 因此在半正态 Q-Q 图上偏离参考直线

### 例题 8.3：混料设计的自由度

**问题**：对于 $p=3$ 的混料设计，为什么 $x_3 = 1 - x_1 - x_2$ 有效降低了维度？

**解**：
约束 $x_1 + x_2 + x_3 = 1$ 意味着三个变量中只有两个是自由的。当 $x_1, x_2$ 在可行域内变化时，$x_3$ 被唯一确定。因此 $p=3$ 的混料设计实际上只有 2 个独立维度（对应三元图中二维三角形的点）。

## 💡 核心要点

1. **半正态图**是识别两水平析因设计中显著效应的直观工具
2. **HiGarrote** 通过贝叶斯收缩自动选择重要变量
3. **多水平设计**可通过 MaxPro 或能量距离准则优化
4. **混料设计**必须满足 $\sum x_i = 1$ 的单纯形约束
5. **SLD** 和 **SCD** 是两种经典的混料设计，**空间填充混料设计**更灵活
